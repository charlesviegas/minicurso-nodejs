Minicurso: Criando Aplicacões Web com NodeJs
=========

Minicurso apresentado no evento <a href="http://www.javaneiros.com.br/">Javaneiros 2014</a>.

## Pré-requisitos
- Node.js

## Execute
```bash
$ node app.js
```

- Abra o navegador na url: <a href="http://localhost:3000>http://localhost:3000</a>.
